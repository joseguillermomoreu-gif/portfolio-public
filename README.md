## 🏗️ Arquitectura Hexagonal

```
src/
├── Domain/          # Lógica de negocio pura (Entities, Value Objects)
├── Application/     # Casos de uso (Services)
└── Infrastructure/  # Adaptadores (Persistence, Http, Controllers)
```
